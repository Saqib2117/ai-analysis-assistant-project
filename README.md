# 📊 AI Data Analysis Assistant

## Overview
An AI-powered data analysis assistant that:
- Loads and analyzes CSV datasets
- Answers natural language questions
- Generates professional charts
- Exports PDF reports

## Tech Stack
- **Backend:** FastAPI
- **Frontend:** Streamlit
- **Analysis:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
- **Deployment:** Docker, Hugging Face, Streamlit
